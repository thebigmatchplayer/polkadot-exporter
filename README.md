# Substratheus

External monitoring for Substrate chains through Prometheus.

## Installation

### Binary

```
sudo wget https://github.com/v-lgns/substratheus/releases/latest/download/substratheus -O /usr/bin/substratheus
chmod a+x /usr/bin/substratheus
```

### Systemd Service Unit

```
sudo wget https://raw.githubusercontent.com/v-lgns/substratheus/master/assets/substratheus.service -O /etc/systemd/system/substratheus.service
```
